# G99 Fast Track Packages

A static HTML site for B4B Renewables, ready for GitHub Pages and Vercel deployment.

## What’s included
- `index.html` — main website file.
- `.gitignore` — keeps the repo clean.
- `deploy_github.sh` — quick Git deploy script for Mac/Linux.
- `deploy_windows.txt` — Git commands for Windows.

## How to use
1. Put all files in the root of your repository.
2. Make sure `index.html` is at the top level.
3. Commit and push to GitHub.
4. Enable GitHub Pages in repository settings, or connect the repo to Vercel.

## GitHub Pages
GitHub Pages serves the site directly from the repository, so just editing and pushing updates will publish changes [web:20][web:21].

## Collaboration
Use branches and pull requests to review changes before merging them into `main` [web:27][web:59].

## Vercel
Import the repository into Vercel for automatic deployments and preview URLs for branches and pull requests [web:4][web:25].
